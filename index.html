    웹 디자인&퍼블리셔 양성과정

저는 디자인을 처음 배우고 실습해보는 웹 퍼블리셔로서, 감각과 기술을 향상시키기 위해 노력하는 학습자입니다. 디자인의 기본 원칙과 웹 디자인에 적용되는 원칙들을 학습하고, 다양한 디자인 툴을 사용하여 시각적인 아이디어를 구체화해나가는 것을 즐깁니다.

저는 디자인에 대한 열정과 앞으로의 성장에 대한 의지를 가지고 있으며, 새로운 도전과 학습을 통해 더 나은 디자이너로 성장하고자 합니다.

LANGUAGES

HTML5

HTML5언어를 통해 홈페이지의 컨텐츠를 입력하고
맥락에 맞는 적절한 태그를 선택하여 웹표준을 준수하여
웹접근성을 높이는 코딩을 할 수 있습니다.

CSS

CSS를 이용해 홈페이지 레이아웃을 구성하며
디자인 및 반응형 media-query를 사용하여
여러 디바이스에 대한 대비가 가능합니다.

JAVA

문서객체를 활용하여 홈페이지 안의 내용 및 효과를 다채롭게 수정가능합니다.
